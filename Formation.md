# Formation React

## Créer un admin

### Présentation du projet

- [Exemple Admin Html](https://horsty80.github.io/responsive-admin-dashboard/)

### Let’s build !

```bash
docker-compose -f docker-compose.dev.yml up
```

```bash
npx create-react-app admin
cd admin
npm start
```

Listes des composants à créer :

- Navigation (Menu)
- Main
  - Topbar
  - CardsBox
  - Details Tables
